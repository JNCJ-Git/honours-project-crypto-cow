# honours-project-crypto-cow
[Code used in Jadon Ng's honours project.]
This code was used in the 2025/26 honours project of Jadon Ng Cheng Jie (2442993) of the University of Dundee.
It is the code used to analyse the given RNASeq datasets of Crypto-infected neonatal calves.
All code used is present in the R markdown document titled "HonoursProject".
